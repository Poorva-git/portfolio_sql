# 📚 Online Bookstore SQL Project

## 📖 Project Overview

This project demonstrates the design and implementation of an **Online Bookstore Database** using **PostgreSQL**. It covers database creation, data import, and SQL-based business analysis using real-world scenarios.

The project showcases SQL skills including filtering, joins, aggregation, subqueries, Common Table Expressions (CTEs), window functions, CASE statements, and views.

---

## 🗂 Database Schema

The database consists of three tables:

- **Books** – Stores book details such as title, author, genre, price, publication year, and stock.
- **Customers** – Stores customer information including name, email, city, and country.
- **Orders** – Stores order transactions linking customers with purchased books.
